MastodonBotについて

#そもそもこれは何#
txtファイルの内容をマストドンに自動投稿するプログラムです。
ターミナルからsample.pyを実行すると、txtの中のランダムな1行が投稿されます。
最初にランダムに並び順を決めて、順番に投稿します。
全部投稿したら並び順を再生成します。
データはpklファイルに保存されます。
つまり、前の投稿と重複しないようになってます。
他のプログラムから呼び出すときはsample.pyをimportしてpost_quote()を実行してください。

#使う前に#
.env_sampleの各設定、サーバーのURL、クライアントキー、クライアントシークレット、アクセストークンをあなたの環境に変更してください。クライアントキー等を見るには、Mastodonのアカウント設定の開発に新規アプリを追加する必要があります。アプリに書き込みの権限の許可のチェックボックスを入れることを忘れないでください。
sample.txtの内容はそのまま投稿内容になります。変更するときは著作権に問題の無いものを使用しましょう。
各ファイル名を変更する場合は、sample.pyの2～4行目のファイル名を対応させてください。ただし、pklは無ければ自動生成されるので、名前は何でもいいです。
ライブラリはdotenvとmastodonを使っていますので、これらはpipでインストールが必要です。

#使用方法#
ターミナルからsample.pyを実行すると、txtの中のランダムな1行が投稿されます。
定期的に実行すれば、txtの内容をサーバーに投稿するbotを作ることができます。
